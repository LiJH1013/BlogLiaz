# 野路子手记设计说明

## 1. 视觉主题

实验感个人刊物。暖纸张底色、超大黑字和橙色标记组成主要识别，布局保持编辑式的不对称与呼吸感。

## 2. 色彩

- Paper：`oklch(96.5% 0.018 78)`，页面底色
- Ink：`oklch(18% 0.015 72)`，正文与深色区域
- Accent：`oklch(72% 0.18 55)`，重点与行动
- Acid：`oklch(91% 0.18 105)`，短句与滚动信息

## 3. 排版

展示字体使用 Arial Black、Aptos Display 与中文系统黑体回退。正文使用 Aptos、系统字体与苹方回退。大标题紧密，中文正文行高 1.7 至 1.9。

## 4. 组件

导航没有容器背景，以细线反馈悬停。文章采用贯穿页面的列表行而非卡片。主要链接使用实心墨色矩形，圆形只用于首页向下入口。

## 5. 空间与 3D

首页的纸张字块使用 CSS 透视、顶面与侧面形成轻量立体感。文章行悬停只做极浅的空间倾斜，不完整旋转，不建立独立 3D 场景。移动端与减少动态模式会停用这些运动。

## 6. 层级

主要依赖背景色切换、排版尺度和留白，不使用通用卡片阴影。便签区使用硬阴影强调纸张实体感，3D 只作为排版层级的补充。

## 7. 内容系统

文章由 `content/posts/*.md` 自动生成，首页、归档、文章导航、RSS 和站点地图共享同一内容源，避免重复维护。

## 8. 约束

不使用渐变文字、玻璃拟态、同构卡片墙和装饰性粒子。不复制参考站品牌、文案或作品。动效只改变 transform 与 opacity。

## 9. 响应式

760px 以下转为单栏，保留 40px 最小点击区域。支持 375px 与 320px 宽度，遵循减少动态效果设置。
