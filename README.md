## 这里写什么

有些内容来自正在做的项目，比如前端、设计和个人网站。有些内容只是生活里的小发现，还不够成为答案，但值得留下。

文章会持续更新。你可以直接访问网站，也可以通过 RSS 订阅新内容。

在文章归档页按 `/` 可以直接开始搜索，按 `Esc` 清空搜索并退出输入。每篇文章都带有目录、阅读进度、分享入口和上一篇、下一篇导航。

## 关于本站

野路子手记没有账户、广告和营销追踪。文章使用 Markdown 保存，网站源码公开在这个仓库中。

如果你发现错字、失效链接或技术问题，欢迎提交 Issue。
